新しいインストール方法:

```
/plugin marketplace add namasugi/claude-toolbox
/plugin install image-gallery@claude-toolbox
```

旧マーケットプレイス(`image-gallery@image-gallery`)からインストール済みの場合は、一度アンインストールして上記で入れ直してください。

プラグインの説明・ドキュメントは [claude-toolbox/plugins/image-gallery](https://github.com/namasugi/claude-toolbox/tree/main/plugins/image-gallery) を参照してください。
